# AviPars

**AviPars** — инструмент для автоматического мониторинга объявлений на Avito.
Позволяет отслеживать новые предложения, получать уведомления и сохранять данные для анализа.

Подходит для:

* перепродажи товаров
* мониторинга цен
* анализа рынка
* поиска выгодных предложений

---

## 🚀 Основные возможности

### 📌 Мониторинг

* Отслеживание новых объявлений в режиме реального времени
* Поддержка нескольких ссылок (до 100 в GUI, без ограничений в CLI)
* Обработка ссылок с уже заданными параметрами

---

### ⚙️ Настройки и фильтрация

* Белый и чёрный список ключевых слов
* Фильтрация по региону
* Исключение конкретных продавцов
* Ограничение по возрасту объявления
* Отслеживание изменений цены
* Игнорирование уже обработанных объявлений

---

### 📊 Сбор данных

* Экспорт результатов в Excel
* Сбор количества просмотров (опционально)
* Получение дополнительных данных (при необходимости)

---

### 📬 Уведомления

* Отправка в Telegram (поддержка нескольких получателей)
* Отправка во ВКонтакте
* Быстрая доставка уведомлений сразу после обнаружения объявления

---

### 🛡 Обход блокировок

* Поддержка прокси (мобильные и серверные)
* Работа через сторонние сервисы (cookies/API)
* Возможность авторизации через аккаунт

---

### ⚡ Производительность

* Быстрая обработка запросов
* Работа в фоновом режиме
* Оптимизация под длительный запуск

---

### 🐳 Режимы запуска

* Графический интерфейс (GUI)
* Консольный режим (CLI)
* Запуск в Docker

---

## ▶️ Быстрый старт

Требуется **Python 3.11+**

Установка зависимостей:

```bash
pip install -r requirements.txt
```

---

### Запуск с интерфейсом (GUI)

```bash
python AvitoParser.py
```

---

### Запуск без интерфейса (сервер / CLI)

```bash
python parser_cls.py
```
