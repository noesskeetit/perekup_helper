## 📲 Настройка уведомлений в Telegram

1. Откройте в Telegram бота BotFather
2. Отправьте команду `/newbot` и задайте имя и username для бота
3. Полученный **token** скопируйте и вставьте в настройки парсера
4. Перейдите в созданного бота и начните с ним диалог
5. Узнайте свой **chat_id** через userinfobot
6. Укажите chat_id в конфигурации (можно добавить несколько, каждый с новой строки)
7. Нажмите кнопку **Test** — при корректной настройке придёт сообщение
8. При необходимости можно указать прокси для Telegram, чтобы повысить стабильность работы

## 📲 Настройка уведомлений во ВКонтакте

1. Создайте сообщество во ВКонтакте → [https://vk.com/groups?w=groups_create_new](https://vk.com/groups?w=groups_create_new)
2. Перейдите в сообщество → **Управление** → **Дополнительно** → **Работа с API**
3. Создайте ключ доступа с правами:

   * отправка сообщений
   * доступ к фотографиям
4. Скопируйте **API-ключ** и вставьте его в настройки парсера
5. Укажите свой **User ID**

   * можно посмотреть здесь: [https://id.vk.com/account/#/personal](https://id.vk.com/account/#/personal)
   * или взять из ссылки вида `vk.com/id1234567` → ваш ID: `1234567`
6. Напишите любое сообщение вашему сообществу в личные сообщения
7. Разрешите сообществу отправлять вам сообщения
8. Нажмите кнопку **Test** — при корректной настройке придёт уведомление

---

## 📌 Примечание

Если используются оба способа уведомлений одновременно:

* сначала сообщения отправляются в Telegram
* затем — во ВКонтакте
